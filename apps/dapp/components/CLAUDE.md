# Dapp Components

Page-level components and shadcn/ui primitives for the dapp.

## Key Files

- `ChatBubble.tsx` — floating support chat widget (bottom-right), calls `/api/chat` for AI responses
- `SaveTemplateModal.tsx` — dialog to name and save invoice creation templates (uses shadcn Dialog)
- `ui/` — shadcn/ui primitives (accordion, alert, avatar, badge, button, card, checkbox, dialog, dropdown-menu, input, label, select, separator, table, tabs, tooltip)

## Patterns

- Page-level components use shadcn imports from `./ui/`
- Package-level components (`@smartinvoicexyz/ui`) use plain Tailwind — never import from `./ui/`
- shadcn config lives at `apps/dapp/components.json`
- Add new shadcn components via `npx shadcn@latest add [component]` from `apps/dapp/`

## Gotchas

- `ui/` directory is auto-generated by shadcn CLI — avoid manual edits to those files
- ChatBubble manages its own message state (not connected to any global store)
